# FairAI – Bias Detection & Correction System

FairAI is an AI-powered platform that detects, explains, and mitigates bias in automated decision-making systems, enabling fair and responsible AI.

---

## Features

- Bias Detection  
  Measures fairness using demographic parity and equal opportunity  

- AI Explanations  
  Provides insights using SHAP and Google Gemini  

- Bias Mitigation  
  Applies Fairlearn techniques to reduce bias  

- Before vs After Comparison  
  Clearly shows improvement in fairness metrics  

---

## Demo Flow

1. Upload dataset or use sample  
2. Detect bias  
3. View metrics and charts  
4. Apply mitigation  
5. Compare results  

---

## Example Output

- Parity Gap: 34% → 5%  
- Equal Opportunity Gap: 29% → 5%  
- Accuracy: 84.7% → 83.1%  

---

## Tech Stack

- Frontend: React  
- Backend: Python (FastAPI)  
- ML: Scikit-learn, Fairlearn, SHAP  
- AI: Google Gemini  
- Cloud: Google Cloud

## Impact

- Reduces biased decisions  
- Improves fairness and transparency
